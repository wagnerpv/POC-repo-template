# Mapa Executivo — Produto

**Este arquivo é um template.** Copie-o para o repositório do produto e preencha com o contexto específico daquele produto.

---

## O que é este documento

O mapa executivo de produto é o equivalente do `mapa-executivo-plataforma.md` para um produto específico.

Enquanto o mapa de plataforma responde "o que estamos construindo como BOX00", o mapa de produto responde "o que este produto é, para quem, e por quê existe".

---

## Diferença entre Plataforma e Produto

**`mapa-executivo-plataforma.md`**
- Vive neste repositório template
- Descreve a BOX00 como plataforma
- Stack canônico, packages, convenções, infra
- Consumido por todos os projetos

**`mapa-executivo-produto.md`** (este arquivo)
- Vive no repositório do produto (ex: brcitas, bitbox00)
- Descreve aquele produto específico
- Público-alvo, proposta de valor, escopo, prioridades
- Consumido apenas pelo time daquele produto

---

## Quando preencher

Ao iniciar um novo projeto a partir deste template, copie este arquivo para o repositório do produto e preencha as seções abaixo com o contexto real do produto.

---

## Seções do Mapa de Produto

### O que é este produto
_(Uma frase. O que é e para quem.)_

### Por que existe
_(O problema que resolve. Por que agora.)_

### Público-alvo
_(Quem usa. Quem paga.)_

### Escopo atual
_(O que está dentro. O que está fora.)_

### Estado atual
_(Em que fase está. O que está funcionando. O que está em construção.)_

### Decisões estratégicas pendentes
_(O que precisa ser decidido antes de avançar.)_

### Perguntas para o fundador
_(Filtros específicos deste produto para decisões do dia a dia.)_
