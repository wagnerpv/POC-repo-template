# sandbox

Ambientes isolados para testes e desenvolvimento local.

## Estrutura

```
sandbox/
└── <environment-name>/
```

Exemplo:

```
sandbox/
├── dev/
├── staging/
└── test/
```

## Uso

Use `sandbox/` para criar ambientes isolados, containers, ou instâncias temporárias de teste sem afetar repositórios oficiais em `apps/` ou `packages/`.
